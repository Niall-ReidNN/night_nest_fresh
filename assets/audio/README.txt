Audio Files for Night Nest Grounding Techniques
================================================

Please add the following audio files to this folder:

1. heartbeat.mp3 - Baby heartbeat sound (60-120 bpm)
2. white_noise.mp3 - White noise/static sound  
3. nicu_ambient.mp3 - Gentle NICU ambient sounds

The app will use these files for audio-based grounding techniques.
You can find royalty-free audio at:
- freesound.org
- pixabay.com/sound-effects/
- YouTube Audio Library

For now, the app will show these options but inform users to add audio files.
